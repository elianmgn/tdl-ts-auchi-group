### Prerequisites

- Follow the steps in the root Readme.
- Create .env in backend folder

### Installation

```bash
$ yarn install
```

### Setup DB

1. Build

```
yarn build
```

2. Create the database, run in the backend terminal.

```
yarn createdb
```

3. Run migrations

```
yarn migrations
```

4. Run seeders

```
yarn seeders
```

5. Run the application using the following command.

```
yarn start
```

6. Default url: `localhost:8081`
